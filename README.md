# SVPWM Simulation

A Python simulation of Space Vector Pulse Width Modulation (SVPWM) for three-phase inverter, developed as part of a renewable energy course project.

## Files

- `SVPWM_simulation_Yixiao.ipynb`: Main simulation notebook (Jupyter/Colab)

## Features

- Three-phase reference signal generation
- Clarke transformation (abc → αβ)
- Sector determination
- Duty cycle calculation (seven-segment SVPWM)
- PWM signal generation
- Inverter output voltage analysis
- FFT spectrum analysis
- Space vector trajectory visualization

## How to run

Open the notebook in [Google Colab](https://colab.research.google.com/) or Jupyter Notebook and run all cells. All required packages (numpy, matplotlib) are available by default.
